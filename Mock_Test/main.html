Steps to complete the challenge

Task 1 :

clean_data

Complete the following operations inside the clean_data function :

The following are the parameters:

        • spark session : - spark

        • input path : - input_path (you can use this parameter to get the CSV file)

  1. Read the CSV data into a data frame.

  2. Drop the rows if it is having null values in any of the columns.

  3. In challenge file, the return statement is already defined, and you need to replace the df with your final output data frame name.


Task 2 :

load_data

Complete the following operations inside the load_data function:

The following are the parameters :

        • Final data frame to load the data : data (final data frame of the following functions)

        • Outpath for the results: outputpath

  1. Write a code to store the outputs to the respective locations.

      • Output files should be a single partition CSV file with header.



Note: The column names (column names are case-sensitive) and order should be same as given in the sample output for each task.



Task 3:

output1

Complete the following operations inside the output1 function:

The following is the parameter :

      • Input data frame for this function : input_df (clean_data final data frame)

  1. From the input_df, fetch the records which contains the following values in the name column.

    • sport, xuv, bmw, jaguar, tata.

Note: The above values might be in Capital letters in the name column.

  1. After fetching the records, get only the records which have the engine CC more than 1500.

Note: Engine column contains numbers and strings.

  1. Order the engine column in descending.

  1. In challenge file, the return statement is already defined. You need to replace the df with your final output data frame name.

Sample output:

name,year,selling_price,km_driven,transmission,max_power,engine

BMW X6 xDrive30d,2013,3750000,56000,Automatic,241 bhp,2993 CC

BMW 6 Series GT 630d Luxury Line,2018,6000000,28156,Automatic,261.4 bhp,2993 CC



Task 4:

output2

Complete the following operations inside the output2 function :

The following is the parameter:

      • Task 1 final data frame : - input_df (you can use this parameter to get the data frame)

  1. In input_df, fetch only the records where the year is greater than 2014.

  2. In the name column, car names are given along with the model name. Keep only first wrod from the name column and exclude the other words.

  3. Based on the name, find the sum of the km_driven column and rename the column to Total_km.

  4. In challenge file the return statement is already defined and you need to replace the df with your final output data frame name.

Sample output :

name,Total_km

Maruti,54281934

Volkswagen,3488777

Lexus,680000



Task 5:

output3

Complete the following operations inside the output3 function:

The following are the parameters :

      • Input data frame for this function : input_df (clean_data final data frame)

  1. From the input_df, fetch the records which are petrol engine and where the selling price greater than 500000.

  2. Create a new column named Quantity, perform the following operation.

    • If the km_driven is less than or equal to 50000, the give the flag as Good.

    • If the km_driven is greater than 50000 and less than or equal to 75000, then give flag as Medium.

    • If the km_driven is greater than 75000 and less than or equal to 100000, then give flag as Low.

    • Otherwise give Not recommened.

  1. In challenge file the return statement is already defined and you need to replace the df with your final output data frame name.

Sample output:

name,year,selling_price,km_driven,fuel,seller_type,transmission,owner,mileage,engine,max_power,seats,Quality

Hyundai i20 1.2 Asta Dual Tone,2018,730000,2388,Petrol,Individual,Manual,First Owner,18.6 kmpl,1197 CC,81.83 bhp,5,Good

Hyundai Verna VTVT 1.6 SX Option,2019,1149000,5000,Petrol,Individual,Manual,First Owner,17.0 kmpl,1591 CC,121.3 bhp,5,Good
